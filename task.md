# Task

# 吳榮交趾陶藝術品牌官網建置專案

## 專案目標
建置對標國際頂級工藝品牌（如 Meissen）的吳榮交趾陶官方網站，將台灣「葉王派系」交趾陶從傳統廟宇飾件提升至國際奢侈品藝術收藏等級。

---

## 一、品牌核心價值與敘事脈絡

### 品牌願景
將交趾陶從「神明看的工作」轉化為「當代收藏家的藝術珍品」。網站敘事需扣合吳榮從「天」（葉王傳承的傳統神聖性）走向「地」（關懷人間情感與當代美學）的轉變。

### 1.1 品牌故事（關於大師頁面）
撰寫 500 字品牌故事，需強調：
- 吳榮童年於雲林斗六磚窯廠旁嬉戲、與黏土共生的感性起點
- 承襲恩師林添木所得之「寶石釉」秘方
- 1980 年代後的變遷：從「半面立體」建築裝飾演化為「3D 全立體化」藝術創作
- 實現「天與地的交會」的藝術突破

### 1.2 四大技術支柱（作為權威標題）
1. **奈米黏土的物理極限**：利用奈米複合材料克服熱漲冷縮，實現大型 3D 作品的結構穩定
2. **黃金淬鍊的胭脂釉色**：胭脂紅（Wu's Carmine）發色自純金（Au），研發出超過 100 種具有折射深度的熟釉色
3. **挑戰重力的動感美學**：以「最小支撐點」支撐龐大量體（如《齊天大聖》），呈現陶瓷罕見的「飄浮感」
4. **跨文化的東方身形**：揉合西方古典美學比例與東方神祇法相，重塑當代陶瓷的優雅輪廓

## 二、視覺語言與 UI/UX 設計規範

### 2.1 設計 Tokens
- **高比例留白（60%+）**：營造美術館般的靜謐感，禁止廉價促銷元素
- **品牌主色：黃金胭脂紅（Wu's Carmine）**：源於黃金的華麗色澤
- **點綴色：冰晶折射藍（Ice-Crackle Blue）**：模擬「冰裂紋」的冷冽與深邃
- **字體**：標題使用優雅襯線體（Serif），內文使用極簡無襯線體（Sans-serif）

### 2.2 沉浸式首頁（Hero Section）
- 使用《齊天大聖》或《雲童》作品
- 呈現「陶瓷重量感」與「腳踩筋斗雲、單點支撐」之間的張力
- 背景運用 WebGL 或微動態效果
- 細微展現釉面「冰裂紋」的物理折射感

### 2.3 冰裂紋深度視覺技術
- 作品展示頁實作「冰裂紋細節縮放」
- 滑鼠懸停時觸發高清細節，展現冰塊般的晶瑩紋理
- 強調光線折射與層次深度

## 三、內容架構與策展策略

### 3.1 策展式導覽結構（作品分類）

#### 1. 傳承經典 (Heritage)
- 作品：《東海龍王》、《觀音騎豺》
- 展現「基本功」如捲鬃、毛流的極致表現，向葉王致敬

#### 2. 當代意象 (Contemporary)
- **重點標註：2020 法國馬賽巡展系列**
- 作品：《相親》、《胭脂水粉》、《出嫁》
- 論述如何將台灣女性情感轉化為國際藝術語言

#### 3. 生命體悟 (Philosophy)
- 核心：《雲童》系列
- 建立「季節滾動時間軸」（Scrolling Timeline）
- 表現從春（幼年繽紛）到冬（生命循環/往生輪迴）的哲學意涵

#### 4. 大型裝置 (Installation)
- 作品：《隱田白鴿》、嘉邑天后宮《千里眼順風耳》
- 展示突破尺寸限制的空間張力

### 3.2 巔峰作《出嫁》專屬解構頁
- 鳳袍上「黃金胭脂紅」的大量應用
- 巴洛克式圓點圖騰的文化揉合
- 鳳凰羽翼上精確的「流釉控制」
- 強調其作為 2020 法國巡展戰略核心的地位
